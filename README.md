# 🏥 도수치료예약 (Dosu Clinic)

> 도수치료 전문 병원을 위한 **환자·예약·직원·통계** 통합 관리 시스템
> Windows 단독 실행형 · 로컬 SQLite · 멀티-PC 동기화 지원

이 리포는 **배포·자동 업데이트 전용** 입니다. 실제 소스 코드는 별도로 관리되며, 이곳에는 완성된 ZIP 과 업데이트 매니페스트만 올라갑니다.

---

## 📋 주요 기능

### 예약 관리
- 일자별 **치료사 × 시간대** 보드 (드래그로 시간·담당자 변경)
- 체외충격파·주사·연골주사 공용 예약 · 도수치료 개인 담당 예약
- 금일 예약 환자 / 취소 목록 / 미니 달력 사이드바
- 원무과 승인(완료 처리) + 낙관적 락으로 다중 PC 동시 수정 안전
- **기간별 집계 & 통계** (v1.2.9+) — 시작일~종료일 + 프리셋 (오늘/7일/이번 달/지난 달/30일)

### 환자 관리
- 8만 명+ 대용량 대응 (서버 검색 + 페이지네이션)
- 엑셀 **데이터변환** — 기존 환자 명단 일괄 불러오기
- 성별·차트번호·연락처·생년월일 자동 인식
- 치료항목별 처방/완료 횟수 자동 카운트

### 문자 발송 (문자나라 연동)
- 예약 안내 · 리마인더 템플릿
- 변수 치환 (`{환자명}`, `{다음예약날짜}` 등)
- 발송 로그 DB 저장
- 한글 인코딩 자동 (v1.2.4+)

### 통계 & 집계
- 기간 요약 5카드 (예약/완료/도수/취소)
- **매출 중심 분석** — 치료항목별·치료사별 예상 매출
- 치료사별 기간 집계 + **인센티브 자동 계산** (% 또는 고정금액)
- **체외충격파 수동 입력** (v1.2.7+) — 집계 탭에서 더블클릭으로 당일 환자 수 직접 입력

### 치료항목 동적 관리
- 도수 30/60/90분 등 시간 항목 자유 추가
- 수가(가격) · 치료사 인센티브 설정
- 활성/비활성 전환 · 표시 여부 제어

### 시스템
- 관리자 비밀번호 보호 · 감사 로그
- 자동 백업 (기본 60분 간격 · 30개 보관)
- 수동 백업/복원 버튼
- DB 무결성 점검 도구
- **네이비 헤더 디자인** (v1.2.5+) · 상단 접속 주소 복사 (v1.2.6+)

## 🔄 자동 업데이트

v1.2.4 부터 **버튼 3클릭으로 최신 버전으로 자동 업그레이드** 됩니다.

### 최초 1회 설정
1. 프로그램 실행 → **관리자 → ≡ 시스템**
2. **"업데이트 매니페스트 URL"** 칸에 입력:
   ```
   https://hu28035036-ux.github.io/clinic-updates/manifest.json
   ```
3. **[URL 저장]** 클릭

### 업데이트 시
1. **[업데이트 확인]** → 새 버전 있으면 변경사항 미리보기
2. **[⬇ 업데이트 다운로드]** → ZIP 자동 다운로드 + SHA256 검증
3. **[✔ 지금 설치]** → 설치 직전 **DB 자동 백업** + 15~20초 재시작

### 안전 장치
- ✅ 설치 직전 **DB 자동 스냅샷** (`clinic_before_update_*.db`)
- ✅ **SHA256 검증** 으로 파일 무결성 확인
- ✅ 실패 시 **자동 롤백** (이전 버전으로 복구)
- ✅ 데이터 폴더(`%APPDATA%\도수치료예약\`)는 업데이트 영향 없음
- ✅ **CSS 캐시 자동 무효화** (v1.2.8+) — 디자인 변경이 업데이트 즉시 반영

## 💻 시스템 요구사항

| 항목 | 권장 |
|---|---|
| OS | Windows 10 / 11 (64-bit) |
| RAM | 4GB 이상 |
| 디스크 | 200MB (프로그램) + DB 크기 |
| 인터넷 | 자동 업데이트 시만 필요 · 평상시 오프라인 동작 |
| 브라우저 | Chrome / Edge / Whale 최신 (프로그램이 로컬 웹서버로 동작) |

**완전 오프라인 환경에서도 정상 작동** — 문자 발송, 자동 업데이트 외의 모든 기능이 로컬에서 동작합니다.

```

├── clinic.db               환자·예약·직원·치료항목 DB (SQLite)

├── schema_version.txt      마이그레이션 버전 기록
└── backups\
    ├── clinic_20260420_120000.db              (자동 백업)
    ├── clinic_before_update_v1.2.x_*.db       (업데이트 직전 자동)
    └── ...
```

⚠️ **프로그램 폴더 업데이트 시 데이터는 전혀 영향받지 않습니다.**

## 📦 설치 방법

### 신규 설치
1. [Releases](https://github.com/hu28035036-ux/clinic-updates/releases) 에서 최신 ZIP 다운로드
2. 원하는 경로에 압축 해제 (예: `C:\Program\도수치료예약\`)
3. `도수치료예약.exe` 더블클릭
4. 바탕화면 바로가기 자동 생성됨 · 브라우저에서 자동 열림

### 기존 설치에서 업그레이드 (수동)
자동 업데이트 사용 못 하는 경우:
1. 프로그램 종료
2. 최신 ZIP 압축 해제 후 **기존 폴더 덮어쓰기**
3. 재실행

## 🛟 문제 해결

### 프로그램이 안 열려요
1. `도수치료예약.exe` 가 있는 폴더 확인
2. 작업 관리자에서 기존 프로세스 있으면 종료
3. 방화벽/백신 차단 확인 (로컬 포트 8000 사용)

### 디자인이 안 바뀌었어요 (업데이트 후)
- **v1.2.8 이상으로 업데이트** — CSS 캐시가 자동 무효화됨
- 그래도 안되면 브라우저 `Ctrl+Shift+R` 강력 새로고침

### 업데이트 후 이상해요
- 자동 롤백이 동작하면 이전 버전으로 복구됨
- 수동 복구: 프로그램 폴더에서 `_internal.old` → `_internal`, `도수치료예약.exe.old` → `도수치료예약.exe` 로 이름 변경

### 데이터가 사라졌어요
- 실수로 `%APPDATA%\도수치료예약\` 폴더를 건드린 경우 외엔 사라질 수 없음
- `backups\` 안의 최신 `.db` 파일로 **복원하기.bat** 실행

### 문자가 깨져서 도착해요
- v1.2.4 이상으로 업데이트
- 문자나라 발신번호 사전 등록 확인
- 광고성 문자는 `(광고)` + 무료 수신거부 번호 필수

### 서브 PC 에서 메인 PC 접속이 안돼요
- 메인 PC 방화벽에서 포트 8000 허용 확인
- 메인 PC 상단 "📋 서브 주소 복사" 로 주소 공유 (v1.2.6+)
- 같은 네트워크(LAN) 에 있는지 확인
